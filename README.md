Una web hecha en clase utilizando PokeAPI para crear:
- una pokedex
- una lista de los habitats de los juegos
- una lista de todos los objetos

Como agregados extra, la pagina incluye:
- Posibilidad de aparecer un Pokémon variocolor / shiny (Con la misma probabilidad que en los juegos)
- Posibilidad de alternar entre sprites o artworks en el apartado de la pokédex
